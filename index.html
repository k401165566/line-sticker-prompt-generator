🟢 LINE 貼圖指令產生器

上傳參考圖 → 選風格 → 選張數 → 勾文字 → 一鍵產生 ChatGPT Images 2.0 提示詞

⛔ 你正在用 LINE 內建瀏覽器，下載 / 長按存圖會失效
解決方式：跳出去用 Chrome 或 Safari 開這個網址
  1. 點畫面 右上角的「⋮」三點 或 「⋯」按鈕
  2. 選 「用其他瀏覽器開啟」 / 「在 Chrome 中開啟」 / 「在 Safari 中開啟」
  3. 跳到正常瀏覽器後一切就會正常運作
⚠️ 開始前先做這個設定（不做的話文字會跑錯字）
  1. 1. 打開 ChatGPT（電腦上 chatgpt.com 或手機 APP 都可以）
  2. 2. 點畫面 最上面那個「ChatGPT」字樣（手機在頂部、電腦在左上），會跳出一個選單
  3. 3. 選 「Thinking」（不要用 Instant，Instant 會把「害羞」寫成「害着」😅）
為什麼一定要切 Thinking？

Instant 是 ChatGPT 預設的快速模式，但寫複雜中文字（例：害羞、撒嬌、萌萌噠）會寫錯， 而且做出來的人物臉會走樣、構圖也常常忽略指令。
Thinking 模式雖然慢一點（生圖約 1-2 分鐘），但**文字、人臉、細節都會正確很多**。 多等那 1 分鐘很值得。

💡 用法：本工具產生的指令一段一段貼到 ChatGPT，每次只貼一段、附上對應照片，等他生圖完再貼下一段。

1️⃣ 上傳參考照片（可多張）

💡 建議上傳 3-10 張不同表情/姿勢的照片，AI 會依每個情緒挑最合適的當素材。例：笑、驚訝、害羞各拍一張 → 比只給一張效果好非常多，不會出現「假笑」「眼睛變白」這類臉部走樣問題。
